Extend the properties panel with the remaining typography controls - font-family, font-weight, line-height, letter-spacing, and text-align. ## Agent Brief

**Category:** enhancement
**Summary:** Add the remaining typography controls to the properties panel, reusing the capture/save pipeline from the walking skeleton.

**Current behavior:**
After the walking skeleton, the Overlay has a working picker, selection box, breadcrumb, and a single font-size control wired through the fingerprint-and-save pipeline. No other typography controls exist.

**Desired behavior:**
The properties panel also exposes font-family, font-weight, line-height, letter-spacing, and text-align. Each control is seeded from the selected element's computed style on selection, previews live on change, and records its change into that element's Patch alongside font-size. text-align is presented as left/centre/right/justify toggle buttons reflecting the element's current value.

**Key interfaces:**
- The properties-panel control pattern established in 0001 - each control writes a CSS property/value into the selected element's `changes` map (keyed by CSS property name) and applies it inline for live preview.
- text-align toggle group - reflects current value, single active state.

**Acceptance criteria:**
- [ ] Panel exposes font-family, font-weight, line-height, letter-spacing, and text-align controls
- [ ] Each is pre-filled from the selected element's computed style on selection
- [ ] Changing any control previews live on the element
- [ ] text-align renders as left/centre/right/justify toggles reflecting current value
- [ ] Each change is recorded into the element's Patch and appears in the saved Batch
- [ ] The Playwright e2e is extended to cover at least one new typography control

**Out of scope:**
- Box and colour controls - margin, padding, colour, background-colour (issue 0003)
- Resize and position-nudge (issues 0004 / 0005)
- A font-family dropdown sourced from the site's `@font-face` declarations (v2)
